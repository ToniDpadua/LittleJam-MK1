# Little Jam MK1
### Say hello to my little jam

![Little Jam MK1](https://raw.githubusercontent.com/SEU_USUARIO/LittleJam-MK1/main/images/logo.png)

ESP32-powered multi-band RF disruptor capable of operating from 300 MHz to 2.4 GHz using:
- 2× CC1101 (sub-1 GHz)
- 2× NRF24L01+PA+LNA (2.4 GHz)
- OLED 0.96" display
- MicroSD logging
- 5× push buttons (for menu navigation)
- Dual 18650 power bank

## Features (planned & in progress)
- Reactive & continuous jamming modes
- Frequency scanning & signal replay
- Deauth / beacon spam 2.4 GHz
- MouseJack & RollJam attacks
- Web interface + OTA updates
- All configurations saved to SD card

## Current status
- Logo & branding → done
- Schematic design → in progress
- Prototype on perfboard → in progress
- Firmware skeleton → in progress

## Warning
This device can interfere with radio communications.  
Use only in controlled environments and in accordance with local laws.
